# git

git branch查看分支

git init -b创建仓库并且创建分支

git branch -m 原名称 新名称，重命名

git branch -d删除合并的

git branch -D没合并的

git checkout分支的切换

git merge 要合并的分支的名称

git checkout -b 分支名称，创建并切换分支

主要分支一般不要

gitignore忽略文件的版本跟踪，排除不需要版本控制的文件

创建一个.gitignore的文件
