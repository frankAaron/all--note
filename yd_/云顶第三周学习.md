## git标签管理

标签方便查找

git tag 标签名，创建标签

git tag查看标签

git tag 标签名 commit对应的号码，可以补标签

git show 标签名 查看标签信息

-a指定标签名，-m指定说明文字，标签的说明

git tag -a 标签名 -m "  "`可以指定标签信息

git push origin 标签名（tags），推送某个（全部）标签到远程

git tag -d 标签名，删除一个本地标签

git push origin :refs/tags/标签名，删除一个远程标签

注：先删本地在远程
